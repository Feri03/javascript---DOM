
Event pada javascript merepresentasikan sebuah kejadian yang terjadi did dalam DOM.
kejadian tersebut bisa dilakukan oleh user, (mouse event, keyboard event, dll.)

ataupun dilakukan secara otomatis oleh API.
(animasi selesai dijalankan, halaman selesai di load, dll)

cara 'mendengarkan' event
    .Event Handler
        .inline HTML attribute
        .Element method
    .addEventListener()
