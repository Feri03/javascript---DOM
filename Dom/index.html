DOM adalah Antarmuka pemrograman berbasis objek yang merepresentasikan dokumen web

DOM membuat seluruh komponen dari halaman web dapat diakses & dimanipulasi

.Komponen : 1. elemen HTML, 2.Atribut, 3.Text, dll

DOM dapat dimanipulasi (dibuat, diubah, dihapus) menggunakan javascript

Tipe-tipe Node

NodeList VS HTMLCollection

kedunya merupakan kumpulan node
struktur datanya mirip array
NodeList dapat berisi node apapun, sedangkan HTMLCollection harus berisi element HTML
HTMLCollection berifat libe sedangkan nodeList tidak.



Struktur Hierarki DOM

  • Root Node
    • node yang menjadi sumber dari semua node lain di dalam DOM
    • document
  • Parent Node
    • node yang berada 1 tingkat di atas node yang lain
    • body adalah parent dari h1, p, p, div dan a
  • Child Node
    • node yang berda 1 tingkat di bawah node yang lain
    • h1 adalah child dari body
